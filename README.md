# my-sql-mini-project
📘 Student Performance Analysis Using MySQL

Project Overview

The Student Performance Analysis project is a mini database system developed using MySQL to analyze student academic performance and attendance records.
This project demonstrates basic to intermediate SQL concepts including database creation, table design, data insertion, and analytical queries.
It is designed as a practical hands-on exercise to strengthen understanding of relational databases and SQL aggregation techniques.

Objectives

Create and manage a MySQL database
Store structured student data
Perform performance analysis using SQL queries
Generate insights using aggregate functions

Technologies Used

MySQL
MySQL Workbench
SQL (DDL, DML, DQL)

SQL Analysis Performed

1️⃣ Average Marks of Students
Calculates overall academic performance using:
AVG() function

2️⃣ Top Performer
Identifies the highest scoring student using:
ORDER BY
LIMIT

3️⃣ Department-wise Average Marks
Analyzes academic performance by department using:
GROUP BY
AVG()

4️⃣ High Attendance Students
Filters students with attendance greater than 85% using:
WHERE condition

📊 Key Concepts Demonstrated

Database creation (DDL)
Data insertion (DML)
Data retrieval (SELECT queries)
Aggregate functions (AVG)
Sorting (ORDER BY)
Filtering (WHERE)
Grouping (GROUP BY)
